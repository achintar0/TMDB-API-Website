# A website created with Django and TMDB-API.
This website's using the **TMDB-API** to display movies and tv-shows, with their details.
It's being developed with Django framework, pure HTML, CSS and JQuery.

Currently the features available are:
* Weekly trending movies display at home page.
* User login and register.
* Movie/TV Show dynamic search bar, made with AJAX.
* Movie/TV Show search sort by popularity.
* Movie/TV Show details page.

# Setup
The pkg requirements are included in **requirements.txt** file.
Setup your .env file with your own API and secret keys.

I've included the templates at project level and static files at application level for personal preferences.
